# 🌤️ Forestry Weather Hub

A personal learning lab and portfolio project built with Python, Flask, SQL, and BeautifulSoup.

## What it does
- Live local weather from my personal weather station in McAllen, TX
- City weather lookup for any Texas city
- NWS weather alerts by IP address location
- NWS weather alerts by state
- Classic Country Music database with fun facts
- Live webcam feed

## Built With
- Python
- Flask
- SQL / SQLite
- BeautifulSoup
- NWS Weather API
- Cloudflare

## About
This project started as a class assignment for ITSE 1402 Computer Programming at South Texas College and grew into a full learning lab. It combines weather data, web scraping, APIs, and a database into one place.
