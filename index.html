Welcome to Ter's Weather Hub

"Welcome to Ter's Weather Hub — a personal project built with Python and Flask, currently being migrated to Rust for better performance. Pulls live weather data and NWS alerts via API, running on a home server. Features include real-time local station weather, city lookups, IP-based alerts, and state-wide NWS alerts. PostgreSQL database and JavaScript frontend improvements coming soon."
